# HW3 可调参数汇总

## 屏幕 & 窗口

| 参数名 | 值 | 含义 |
|--------|----|------|
| `SCR_WIDTH` | 800 | 窗口宽度（像素） |
| `SCR_HEIGHT` | 600 | 窗口高度（像素） |

---

## 摄像机

| 参数名 | 值 | 含义 |
|--------|----|------|
| `cameraPos` | (0, 5, 10) | 自由视角初始位置 |
| `cameraFront` | normalize(0, -5, -10) | 自由视角初始朝向 |
| `fov` | 45.0° | 视野角度 |
| `thirdPersonDistance` | 2.0 | 第三人称摄像机与企鹅的距离 |
| `thirdPersonHeight` | 2.0 | 第三人称摄像机高度偏移 |
| `thirdPersonPitch` | -15.0° | 第三人称俯视角度 |
| `sensitivity`（第三人称） | 0.2 | 第三人称鼠标旋转灵敏度（度/像素） |
| `sensitivity`（自由视角） | 0.1 | 自由视角鼠标旋转灵敏度（度/像素） |
| `cameraSpeed` | 160 × 4 ÷ 5 × dt | 自由视角 WASD 移动速度（原1/5） |
| 第三人称俯仰范围 | ±60° | 第三人称摄像机上下转动限制 |
| 自由视角俯仰范围 | ±89° | 自由视角摄像机上下转动限制 |

## 光照

### 环境光

| 参数名 | 值 | 含义 |
|--------|----|------|
| `ambientDay` | 0.6 | 环境光强度（固定值，T 键不改变） |

### 方向光（太阳）

| 参数名 | 值 | 含义 |
|--------|----|------|
| `lightBrightness` | 0.7（初始） | 当前方向光亮度 |
| `dirLightDayBrightness` | 0.6 | 日间模式方向光亮度（T 键切换） |
| `dirLightNightBrightness` | -0.5 | 夜间模式方向光亮度 |
| `dirLightDirection` | (-0.05, -0.3, -0.1) | 方向光方向向量 |
| 亮度下限 | -2.0 | 方向光最低允许亮度 |

### 聚光灯

| 参数名 | 值 | 含义 |
|--------|----|------|
| `spotlightBrightness` | 2.0（初始） | 聚光灯亮度 |
| `spotlightAutoThreshold` | 0.2 | `lightBrightness` 低于此值自动开启聚光灯 |
| `spotlightInitialBrightness` | 3.0 | 聚光灯自动开启时的初始亮度 |
| `spotlight1Pos` | (-5.89, 2.58, 0) | 聚光灯 1 位置 |
| `spotlight1Dir` | (1.0, -1.0, 0) | 聚光灯 1 方向 |
| `spotlight1Color` | (1, 1, 1) | 聚光灯 1 颜色（RGB） |
| `spotlight2Pos` | (5.89, 2.58, 0) | 聚光灯 2 位置 |
| `spotlight2Dir` | (-1.0, -1.0, 0) | 聚光灯 2 方向 |
| `spotlight2Color` | (1, 1, 1) | 聚光灯 2 颜色（RGB） |
| `spotlightCutoff` | cos(90°) = 0 | 聚光灯内锥角（90°） |
| `spotlightOuterCutoff` | cos(180°) = -1 | 聚光灯外锥角（180°） |
| 聚光灯亮度范围 | [0.0, 5.0] | 聚光灯允许的亮度范围 |

### 亮度调节步长

| 按键 | 参数名 | 值 | 含义 |
|------|--------|----|------|
| E ↑ / Q ↓ | 方向光步长 | 0.05 | 每按一次方向光亮度增减量 |
| P ↑ / O ↓ | 聚光光步长 | 0.05 | 每按一次聚光灯光亮度增减量 |

## 足球

| 参数名 | 值 | 含义 |
|--------|----|------|
| `soccerPos`（初始/重生） | (0, 1.80, 0.09) | 足球初始位置 |

| `CEILING_LEVEL` | 5.0 | 足球天花板反弹高度 |
| 墙壁反弹系数 | -0.5 | XZ 撞墙速度反转比例 |
| 地面反弹系数 | -0.6 | 落地弹跳速度保留比例 |
| 天花板反弹系数 | -0.3 | 撞天花板速度保留比例 |
| 速度死区 | 0.02 | 水平速度低于此值时归零 |
| 足球渲染缩放 | 0.04 | 模型大小 |

## 障碍物（Craft ×4）

| 参数名 | 值 | 含义 |
|--------|----|------|
| `CRAFT_OSCILLATION_AMPLITUDE` | 3.0 | 横移幅度（半幅，单位） |
| `CRAFT_OSCILLATION_FREQUENCY` | 1.5 | 横移频率（周期/秒） |
| `CRAFT_SELF_ROTATION_SPEED` | 75°/秒 | 障碍物自转速度（原60°/秒） |
| `CRAFT_COLLISION_RADIUS` | 0.2 | 企鹅碰触障碍物扣命的判定距离 |
| `CRAFT_SCALE` | 0.025 | 障碍物模型缩放（原0.02） |
| 接近检测距离 | 1.0 | 企鹅距离 < 1.0 切换为 `spacecraftTexture`（接近预警） |
| `crafts[0].basePos.z` | -2.0 | 第 1 排障碍物 Z 坐标 |
| `crafts[1].basePos.z` | -0.667 | 第 2 排障碍物 Z 坐标 |
| `crafts[2].basePos.z` | 0.667 | 第 3 排障碍物 Z 坐标 |
| `crafts[3].basePos.z` | 2.0 | 第 4 排障碍物 Z 坐标 |

### 障碍物纹理切换逻辑

| 状态 | 条件 | 纹理 |
|------|------|------|
| 默认 | 企鹅距离 ≥ 1.0 且未碰撞 | `ringTexture` |
| 接近预警 | 企鹅距离 < 1.0 | `spacecraftTexture` |
| 碰撞 | `hit == true` | `craftRedTexture` |
